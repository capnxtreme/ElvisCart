# 🎸 ElvisCart - Rockabilly Kart Racing

A pseudo-3D kart racing game with a 1950s rockabilly twist!

## 🏁 Quick Start

```bash
# Start the game
npm run dev
# Open http://localhost:8080
```

## 🎮 Controls

- **↑** - Accelerate
- **← →** - Steer  
- **SPACE** - Drift
